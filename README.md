# Поділ сум по картах (веб-кнопка)

Маленький веб-застосунок (Flask): відкриваєш сторінку → тиснеш **«Запустити поділ
сум по картах»** → скрипт розкидає суми партнерів по картах зарплат у Google
Таблиці (лист «Розкидати карти») і показує зведення + лог.

Логіка розподілу: кожен партнер платить **на одну карту**, якщо його сума туди
влазить; великий партнер — максимум на дві (3+ лише коли він сам більший за дві
найбільші зарплати разом). Суми мають сходитись 1:1.

## Запуск на Replit

1. **Create Repl → Import from GitHub** → `vaadim123/claude_privat`, папка
   `rozkidaty_karty`. (Або скопіюй усі файли в новий Python-Repl.)
2. Додай креди Service Account:
   - **Рекомендовано:** Replit → **Secrets (🔒)** → ключ `GOOGLE_CREDENTIALS`,
     значення = увесь вміст `credentials.json` одним блоком.
   - Або поклади файл `credentials.json` у корінь Repl.
3. Service Account має мати доступ **Редактор** до таблиці.
4. Тисни **Run** → відкриється webview зі сторінкою та кнопкою.

## Локально

```bash
pip install -r requirements.txt
python3 main.py            # потрібен credentials.json поруч
# відкрий http://localhost:8080
```

## Файли

- `main.py` — Flask-сервер + логіка розподілу (`/` сторінка, `/run` запуск)
- `templates/index.html` — сторінка з кнопкою
- `requirements.txt`, `.replit`, `.gitignore`
