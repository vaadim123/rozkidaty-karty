      Як налаштувати (один раз)
      
1. Google Cloud Console → APIs & Services → Enable «Google Sheets API».

         2. Credentials → Create credentials → OAuth client ID → тип Web application.

         3. У «Authorized JavaScript origins» додай https://vaadim123.github.io.

         4. Скопіюй Client ID у поле вище (зберігається в браузері).

         5. Таблицю має бути доступно твоєму Google-акаунту на редагування.
